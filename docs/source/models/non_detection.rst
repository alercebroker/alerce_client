Non Detection Response
=========================

.. list-table:: Non Detection Model
  :widths: 20 10 70
  :header-rows: 1

  * - Name
    - Type
    - Description
  * - oid
    - integer
    - Object identifier
  * - band
    - integer
    - Filter/band identifier (1=g; 2=r; 3=i)
  * - mjd
    - number
    - Modified Julian Date of the observation
  * - diffmaglim
    - number
    - 5-sigma limiting magnitude in difference image [mag]
